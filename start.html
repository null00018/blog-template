📡 ブログテンプレートをはじめよう

このページでは、あなた自身のブログを GitHub 上だけで作り、記事投稿もワンクリックでできる方法を紹介します。

🛠 ステップ 1: GitHub アカウントを作成

  1. GitHub サインアップページにアクセス
  2. ユーザー名・メールアドレス・パスワードを入力
  3. ロボットではないことを確認し、[Create Account] をクリック
  4. 無料プラン（Free）を選んで次へ

📦 ステップ 2: テンプレートをコピー

下のボタンを押すと、あなた専用のブログフォルダ（リポジトリ）が作成されます。

🚀 Use This Template

ボタンを押すと次のような画面になります：

  1. Repository name：好きなブログ名を入力（例：my-blog）
  2. Description：（任意）ブログの説明を入力できます。空欄でもOK。
  3. Visibility：「Public（公開）」にチェック（そのままでOK）
  4. 確認後、右下の Create repository ボタンをクリック！

これであなた専用のブログテンプレートが作成されます。次に、GitHub Pages を有効にしましょう。

🌍 ステップ 3: GitHub Pages を有効に

  1. 作成した自分のリポジトリにアクセス
  2. 右上の「Settings」 → 左側の「Pages」へ
  3. "Branch: main / (root)" を選んで「Save」

約30秒〜1分後に、あなた専用のブログが公開されます。
